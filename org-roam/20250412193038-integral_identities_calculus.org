#+title:Integral identities, calculus

* Some useful ones

** Splitting up integrals

If \(a \leq b \leq c\), then

\[\int_a^c f(x)dx = \int_a^b f(x)dx + \int_b^c f(x)dx\]

If \(b \geq a\)

\[\int_a^b f(x)dx = \int_0^b f(x)dx - \int_0^a f(x)dx\]
