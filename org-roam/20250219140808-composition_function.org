#+title:Composition, function

Function composition, written $f \circ g$ for some $f$ and $g$, is defined as $(f \circ g)(x) = f(g(x))$.
If
\begin{align*}
f &: A \to B
\\
g &: B \to C
\end{align*}

then naturally

\begin{align*}
\text{Range}(g) \subseteq \text{Dom}(f)
\end{align*}
